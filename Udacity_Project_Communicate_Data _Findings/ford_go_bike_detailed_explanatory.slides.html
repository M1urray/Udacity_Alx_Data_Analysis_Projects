Dataset Overview¶

This data set includes information about individual rides made in a bike-sharing system covering the greater San Francisco Bay area. The data set has been stored as a pandas dataframe, It has 16 columns and 183412 rows. The features cover 3 main areas:

  1. trip duration
  2. station information
  3. member information
